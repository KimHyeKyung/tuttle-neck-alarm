# 🐢 거북목 알리미

> 30분마다 거북이가 화면 아래에서 기어나와 거북목을 조심하라고 알려주는 프로그램

![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey)
![License](https://img.shields.io/badge/License-MIT-green)

---

## 📺 미리보기

30분마다 화면 오른쪽 아래 작업표시줄 위에서 🐢 가 기어나와 메시지를 보여주고 돌아갑니다.

```
거북목 피세요! 목을 뒤로 당기세요 🐢
```

---

## 🚀 실행 방법

1. `거북목알리미.exe` 다운로드
2. 더블클릭해서 실행

**Python 설치 불필요 — exe 파일 하나로 바로 실행됩니다!**

---

## 🖥️ 기능

- 30분마다 🐢 가 화면 아래에서 등장
- 말풍선으로 거북목 경고 메시지 표시
- 배경 투명 처리 (거북이만 보임)
- 작업표시줄 트레이 아이콘으로 관리
  - 우클릭 → **종료** 로 프로그램 종료
  - 우클릭 → **지금 당장 나타내기** 로 즉시 호출

---

## 🔄 시작프로그램 등록 (선택)

컴퓨터를 켤 때마다 자동으로 실행하려면:

1. `Win + R` → `shell:startup` 입력 → 엔터
2. `거북목알리미.exe` 우클릭 → **"바로 가기 만들기"**
3. 생성된 바로가기를 열린 폴더에 붙여넣기

---

## ⚠️ 보안 경고 관련

처음 실행 시 Windows Defender에서 경고창이 뜰 수 있어요.

**"추가 정보" → "실행"** 을 클릭하면 정상 실행됩니다.
